# ZeroLink

基于 Shizuku 和 WebRTC 的 Android 远程控制应用。

## 功能特性

- 局域网设备发现
- WebRTC 视频流传输
- 远程输入注入
- 基于 Shizuku 的系统权限

## 项目结构

- `app-controlled` - 受控端应用
- `app-controller` - 控制端应用
- `core-domain` - 领域层
- `core-infrastructure` - 基础设施层
- `core-shizuku-server` - Shizuku 服务
- `lib-common` - 公共库
